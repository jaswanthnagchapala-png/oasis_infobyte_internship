# Oasis Infobyte Cybersecurity Internship

## Overview

This repository contains the cybersecurity tasks that I successfully completed during my Oasis Infobyte Cybersecurity Internship. The internship provided hands-on experience with industry-standard cybersecurity tools and helped me understand practical concepts related to network security, vulnerability assessment, firewall management, and network traffic analysis.

All tasks were performed in a Kali Linux environment. Throughout the internship, I installed, configured, and used various cybersecurity tools to perform security assessments, analyze network behavior, and document my findings. Each task folder contains screenshots, observations, commands executed, and conclusions based on the activities performed.

---

## Task 1 – Network Port Scanning with Nmap

### Objective

To perform network reconnaissance and identify active hosts, open ports, and available services using Nmap.

### Work Completed

* Installed and verified Nmap installation.
* Performed basic network scanning on localhost.
* Executed service detection scans.
* Analyzed port status and scan results.
* Documented findings with screenshots.

### Skills Gained

* Network Reconnaissance
* Port Scanning
* Service Enumeration
* Network Mapping

### Outcome

Successfully performed network scanning and gained practical experience in identifying network services and understanding port states.

## Task 2 – Firewall Configuration Using UFW

### Objective

To configure and manage firewall rules using UFW (Uncomplicated Firewall) in Kali Linux.

### Work Completed

* Allowed SSH traffic through port 22.

* Reviewed and managed firewall rules.
* Documented results with screenshots.

### Skills Gained

* Firewall Configuration
* Traffic Filtering
* Access Control
* Linux Security Management

Successfully configured firewall policies and learned how to secure systems by controlling network access.
